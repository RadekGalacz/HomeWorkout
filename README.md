# HomeWorkout Aplikace

Tato mnou vytvořená webová aplikace slouží k vytváření a správě denních cvičebních plánů. Umožňuje uživatelům přidávat cviky, označovat je jako odcvičené, a spravovat cvičené partie a cviky v administrační sekci. Aplikaci plánuji rozšířit o přístupová práva a možná další funkčnost. Aplikace demonstruje práci jak s frontendem, tak s backendem.

## ✨ Hlavní funkce

- Vytvoření a správa denního plánu cvičení
- Filtrování cviků podle stavu (odcvičeno / neodcvičeno)
- Editace nebo smazání denního plánu
- Možnost označit cvik jako odcvičený nebo ho smazat
- Přidání nových cviků do plánu
- Správa cvičení a partií v administrační sekci
  - Přidání, editace nebo smazání partií
  - Vytvoření, editace nebo smazání cviků
  - Uzamčení položek, které nelze mazat ani editovat
  - Tlačítka pro přidávání položek jsou ve výchozím stavu deaktivována a aktivují se až po vyplnění všech povinných polí.

## 🛠️ Použité technologie

- React – JavaScript framework pro vytváření uživatelského rozhraní
- Axios – pro komunikaci s API
- CSS – pro styling aplikace
- Vite – pro vývoj a optimalizaci aplikace
- ASP.NET Core Web API – pro backend a API
- Azure – pro hostování backendu
- MonsterASP.NET – pro hostování MSSQL databáze
- Osobní webové stránky – pro hostování frontendové části aplikace
